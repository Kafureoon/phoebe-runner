# 菲比快跑

一个纯前端像素风横版跑酷小游戏。项目只使用当前 `assets` 目录里的本地素材，可以直接部署为静态站点。

## 本地运行

直接双击 `index.html` 可以运行。更推荐在项目目录启动一个本地静态服务：

```powershell
cd D:\AIfiles\phoebe-runner
python -m http.server 5175 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:5175/
```

## 操作

- `空格` / `↑` / 点击画面：短按小跳，按住一小段时间会升级为大跳，也可开始游戏。
- `↓` / `S`：下蹲躲避空中障碍。
- `R`：重新开始。

## 当前内容

- 使用用户提供的 5 张素材表重新裁切角色和场景。
- 使用画图生成的新障碍物图集补充四个场景的障碍池，源图保存在 `assets/generated_sources`。
- 菲比包含跑步、跳跃、下蹲和撞击状态。
- 场景包含水岸庭院、花庭小径、月夜海岸、黑海岸遗迹，约每 1500 分切换一次，切换动画使用固定时长，不跟障碍速度绑定。
- 障碍物会按当前场景切换，不再混用外部素材；2000 分后会混入从天上掉到路面的坠落障碍，4000 分后会混入必须大跳才能越过的大障碍，6000 分后会同时出现小型弹跳和大型弹跳障碍，都需要下蹲通过。
- 障碍物速度每 500 分提升一级，背景和跑道保持固定视觉速度，避免场景越跑越乱。
- 地图会随机刷新护盾道具，第一件会在 500-1000 分内出现；拾取后 30 秒内可免疫一次死亡，并带有玻璃球护盾特效。
- 最高分保存在浏览器本地存储中。

## 部署

把整个 `phoebe-runner` 文件夹上传到服务器的静态网站目录即可。入口文件是 `index.html`，不需要后端服务。
